First Weeks at CodeFactory

I've been learning about web development at CodeFactory. Here's a summary of what I've learned so far and what I will learn in the upcoming weeks:

Git and GitHub

  • Understanding version control and the necessity of Git.
  • Installing and configuring Git
  • Making commits and following best practices and standards.
  • Working with GitHub repositories
  • Cloning projects from GitHub to the local machine.

HTML

  • Using HTML tags to create and structure websites.
  • Creating a simple HTML document and understanding its components.
  • Formatting text with headings and paragraphs.
  • Incorporating images and links into web pages.
  • Distinguishing between block-level and inline elements, using div and span.

CSS

  • Understanding the anatomy of Cascading Style Sheets (CSS).
  • Using internal, external, and inline styles.
  • Applying CSS selectors including element, ID, and class selectors.
  • Exploring the box model and its components: dimensions, margin, padding, and border.
  • Implementing different positioning methods: static, fixed, absolute, sticky, and relative.
